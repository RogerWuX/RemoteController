# RemoteController
Simple remote controller using webrtc to exchange framebuffer and control message.

## Platform
- Ubuntu 20.04

## Dependency
- Protobuf 3.21.5
- OpenSSL(required by asio and qt)
### client
- Qt 5.12.8
- Webrtc-native branch-heads/5280
- spdlog 1.10.0
### match server
- asio 1.24.0
